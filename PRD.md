# 원재 연산앱 PRD

## 1. 제품 한 줄 정의
초등학교 3학년 아동이 3자리 덧셈/뺄셈의 자리올림·자리내림 유형을 **구분해서 반복 숙달**할 수 있게 돕는 모바일/웹 학습 앱.

## 2. 배경
원재가 현재 필요한 것은 단순한 연산 문제풀이가 아니라, 다음과 같은 **유형별 숙달**이다.
- 3자리 덧셈
- 받아올림 2번
- 받아올림 3번
- 받아내림 1번
- 받아내림 2번

일반 연산앱은 문제를 섞어서 내는 경우가 많아, 아이가 어떤 유형에서 헷갈리는지 분리해서 훈련하기 어렵다.
이 제품은 문제를 많이 푸는 것보다, **헷갈리는 유형을 정확히 구분하고 반복하는 경험**을 우선한다.

## 3. 목표
### 핵심 목표
- 원재가 자리올림/자리내림 유형을 스스로 구분해 연습할 수 있게 한다.
- 특정 유형만 반복해서 훈련할 수 있게 한다.
- 오답을 다시 복습하며 약한 유형을 보완하게 한다.

### 부가 목표
- 학부모가 어떤 유형이 약한지 쉽게 확인할 수 있게 한다.
- 모바일에서도 큰 글자와 단순한 흐름으로 부담 없이 사용할 수 있게 한다.
- 짧은 세션(5~10분) 안에 연산 연습이 가능하게 한다.

## 4. 대상 사용자
### 1차 사용자
- 초등학교 3학년 아동 (원재)

### 2차 사용자
- 학부모 또는 보호자
  - 어떤 유형이 약한지 확인
  - 오늘 연습할 유형 선택 보조

## 5. 핵심 문제 정의
1. 아이가 자리올림/자리내림 유형을 섞어서 인지해 헷갈린다.
2. 문제를 많이 풀어도 어느 유형에서 실수하는지 분리하기 어렵다.
3. 피드백이 너무 길거나 복잡하면 학습 집중이 깨진다.
4. 모바일에서 숫자와 세로셈 구조가 충분히 명확하지 않으면 학습 효과가 떨어진다.

## 6. MVP 범위
### 포함
- 유형 선택
  - 3자리 덧셈
  - 받아올림 2번
  - 받아올림 3번
  - 받아내림 1번
  - 받아내림 2번
- 문제 생성 엔진
  - 각 유형 규칙을 만족하는 문제 생성
- 문제 풀이 화면
- 정답/오답 판정
- 간단한 피드백
- 결과 화면
- 오답만 다시 풀기
- 유형별 정확도 저장
- 오늘의 추천 연습

### 제외
- 멀티 유저
- 회원가입/로그인
- 소셜/랭킹
- 복잡한 게임화 시스템
- 음성 피드백
- AI 풀이 해설
- 교사용 대시보드 고도화

## 7. 핵심 사용자 흐름
### 흐름 A. 유형 선택 연습
1. 홈 진입
2. 오늘 추천 또는 특정 유형 선택
3. 10문제 풀이
4. 결과 확인
5. 부족한 유형 다시 연습

### 흐름 B. 오답 복습
1. 결과 화면에서 오답만 다시 풀기 선택
2. 틀린 문제 재풀이
3. 재결과 확인

### 흐름 C. 보호자 확인
1. 홈 또는 기록 화면에서 유형별 정확도 확인
2. 약한 유형 중심으로 다음 학습 선택

## 8. 기능 요구사항
1. 사용자는 연산 유형을 선택할 수 있어야 한다.
2. 앱은 선택된 유형 규칙에 맞는 문제만 생성해야 한다.
3. 문제는 큰 숫자와 명확한 세로셈 구조 또는 유사 구조로 보여야 한다.
4. 사용자는 답을 입력하고 제출할 수 있어야 한다.
5. 앱은 즉시 정답/오답을 알려줘야 한다.
6. 오답 시 너무 긴 설명 대신 자리 힌트 중심 피드백을 제공해야 한다.
7. 세션 종료 후 유형별 정답률을 보여줘야 한다.
8. 사용자는 오답만 다시 풀 수 있어야 한다.
9. 앱은 유형별 누적 정답률과 최근 학습 기록을 로컬에 저장해야 한다.
10. 홈 화면에서 오늘 추천 연습 또는 약한 유형을 제안할 수 있어야 한다.

## 9. 비기능 요구사항
- 큰 글자와 단순한 인터페이스
- 모바일 우선
- 빠른 진입
- 로컬 저장 기반
- 짧은 세션에 적합한 속도
- 아이가 혼자 써도 헷갈리지 않는 UI

## 10. 성공 기준
- 원재가 원하는 유형만 골라서 10문제 이상 반복 학습할 수 있다.
- 각 문제 유형이 규칙대로 정확히 생성된다.
- 결과 화면에서 약한 유형을 쉽게 알 수 있다.
- 오답 복습이 자연스럽게 이어진다.
- 학부모가 봐도 어떤 유형이 약한지 분명히 보인다.

## 11. 첫 번째 MVP 이후 확장 후보
- 세로셈 도움 힌트 on/off
- 단계별 힌트
- 학습지 출력 / PDF
- 학부모용 주간 요약
- 문제 난이도 세분화
- 혼합 테스트 모드 고도화
