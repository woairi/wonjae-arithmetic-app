# CHANGELOG

## 2026-03-28

- 기록 삭제/초기화 전 확인 단계 UI 추가
- 기록 삭제 뒤 남은 `history` 기준으로 누적 정확도와 요약을 다시 계산하는 저장 로직 추가
- 결과 화면 상단 `바로 이어서` 추천 카드 추가
- 홈과 최근 학습 기록 화면에 `최근 7일 요약` 카드 추가
- 최근 7일 연습 횟수 / 가장 많이 푼 유형 / 가장 많이 틀린 유형 / 평균 정확도 집계 추가
- 홈과 결과 화면 추천 카드에 짧은 `이유:` 문구 추가
- 최근 7일 요약 계산 검증 테스트 추가

### Changed
- 홈 화면에 보호자 요약과 분리된 `기록 정리` 진입 카드 추가
- 문제 풀이 화면에서 정답 뒤 잠깐 후 자동으로 다음 문제/결과로 이어지도록 조정
- 문제 풀이 화면 입력칸과 제출 뒤 다음 버튼의 포커스를 다시 정리
- 결과 화면에서 추천된 다음 연습을 첫 카드에서 바로 시작할 수 있게 흐름 단축
- 홈 추천이 최근 7일 약점, 최근 2회 이상 연속 성공, 혼합 연습 균형을 함께 보고 자연스럽게 바뀌도록 조정
- 어려운 유형이 최근 연속으로 낮은 정확도를 보이면 한 단계 쉬운 유형으로 되돌아가도록 추천 분기 보강
- 혼합 모드 뒤에는 여러 유형 재시도와 단일 약한 유형 추천을 분리해서 보여주도록 조정
- 홈 화면 카드 구성을 최근 연습 횟수 단일 카드 대신 주간 요약 중심으로 재배치
- 최근 학습 기록 화면 상단에서도 최근 7일 요약을 함께 보이도록 조정

### Notes
- 기록 정리 기능은 로컬 `history`를 기준으로 `stats`를 다시 계산해 특정 세션 삭제 뒤에도 누적 정확도가 남은 기록과 맞도록 유지한다.
- 최근 7일 요약은 `history`에 저장된 최근 20개 세션 중 최근 7일 안에 끝난 세션만 집계한다.
- 평균 정확도는 최근 7일 세션의 전체 정답 수 / 전체 문항 수 기준으로 계산한다.

### Changed

- 문제 화면에서 세션 모드와 현재 문제의 실제 유형을 분리해서 보이도록 조정
- 자리값 보드 배경, 자리 라벨, 숫자 칸 대비를 높여 초3 기준 가독성 보강
- 결과 화면에서 이번에 나온 실제 유형, 틀린 실제 유형, 틀린 문제별 실제 유형을 함께 보이게 조정
- 결과 화면에서 이번 세션의 약한 실제 유형 1~2개가 먼저 보이도록 강조 영역 추가
- 혼합 세션 결과도 기존 실제 유형별 누적 통계 구조에 반영되도록 저장 로직 정리
- 오답 다시 풀기에서 혼합 세션의 실제 문제 유형 정보가 유지되도록 세션 재구성 로직 보강
- 혼합 세션에서도 틀린 실제 유형만 골라 새 문제 10개를 다시 만들 수 있도록 재연습 로직 확장
- 오답 힌트를 자리 단위의 더 짧고 구체적인 문구로 조정
- 문제 화면에 `세로셈 힌트 보기 / 숨기기` 토글과 자리 열 강조를 추가
- 문제 화면의 세로셈 힌트를 `숨김 / 먼저 / 직접` 3단계 구조로 확장
- `먼저` 단계는 시작할 자리 하나만, `직접` 단계는 실제 올림/내림 자리를 더 직접적으로 보여주도록 보드와 문구를 조정
- 힌트를 켰을 때 자리값 보드가 더 또렷하게 읽히도록 라벨/숫자 칸 대비와 관련 자리 강조를 보강
- 오답 제출 뒤에는 관련 자리 열을 한 단계 더 강하게 표시하도록 조정
- 최근 연습 횟수, 가장 약한 유형 1~2개, 유형별 정답률 요약이 홈 화면에 최소 카드 형태로 보이도록 조정
- 홈 화면의 최근 기록 미리보기와 `최근 학습 기록` 진입 버튼을 추가해 보호자 요약과 최근 세션 흐름을 분리
- 결과 화면 CTA를 짧게 줄이고 틀린 문제 재학습 흐름을 더 바로 보이게 조정
- 결과 화면에 이번 세션 기준 한 줄 추천 카드와 연결된 대표 CTA를 추가
- 혼합 모드 / 단일 유형 / 만점 세션 분기에 따라 다음 추천 연습 문구가 자연스럽게 바뀌도록 조정
- `.gitignore`에 `.omx/`와 환경 흔적 제외 규칙 추가
