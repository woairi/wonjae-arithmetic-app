# CHANGELOG

## 2026-03-27

### Added
- 모바일 우선 정적 웹 앱 골격 추가
- 홈, 유형 선택, 문제 풀이, 결과 화면 구현
- 10문제 세션 흐름 및 오답 다시 풀기 구현
- 유형별 문제 생성, 분석, 검증 엔진 구현
- `전체 유형 랜덤` 혼합 세션 모드 추가
- 결과 화면의 `가장 약한 유형 새 문제 10개` / `틀린 유형만 다시` 복습 CTA 추가
- 홈 화면 보호자용 최소 요약 카드 추가
- 홈에서 이동 가능한 `최근 학습 기록` 전용 화면 추가
- `localStorage` 기반 정확도 / 최근 학습 기록 저장
- 정적 빌드 스크립트, 로컬 서버 스크립트, 엔진 테스트 추가
- Stitch 검토 메모 문서 추가

### Changed
- 홈, 유형 선택, 문제 풀이, 결과 화면 문구를 더 짧고 쉬운 표현으로 정리
- 덧셈 / 뺄셈 유형 카드를 그룹 색상과 규칙 칩으로 다시 구성
- 문제 화면에 현재 유형, 문제 번호, 식 표시를 앞세워 읽기 흐름 개선
- 문제 화면에서 세션 모드와 현재 문제의 실제 유형을 분리해서 보이도록 조정
- 자리값 보드 배경, 자리 라벨, 숫자 칸 대비를 높여 초3 기준 가독성 보강
- 결과 화면에서 이번에 나온 실제 유형, 틀린 실제 유형, 틀린 문제별 실제 유형을 함께 보이게 조정
- 결과 화면에서 이번 세션의 약한 실제 유형 1~2개가 먼저 보이도록 강조 영역 추가
- 혼합 세션 결과도 기존 실제 유형별 누적 통계 구조에 반영되도록 저장 로직 정리
- 오답 다시 풀기에서 혼합 세션의 실제 문제 유형 정보가 유지되도록 세션 재구성 로직 보강
- 혼합 세션에서도 틀린 실제 유형만 골라 새 문제 10개를 다시 만들 수 있도록 재연습 로직 확장
- 오답 힌트를 자리 단위의 더 짧고 구체적인 문구로 조정
- 문제 화면에 `세로셈 힌트 보기 / 숨기기` 토글과 자리 열 강조를 추가
- 힌트를 켰을 때 자리값 보드가 더 또렷하게 읽히도록 라벨/숫자 칸 대비와 관련 자리 강조를 보강
- 오답 제출 뒤에는 관련 자리 열을 한 단계 더 강하게 표시하도록 조정
- 최근 연습 횟수, 가장 약한 유형 1~2개, 유형별 정답률 요약이 홈 화면에 최소 카드 형태로 보이도록 조정
- 홈 화면의 최근 기록 미리보기와 `최근 학습 기록` 진입 버튼을 추가해 보호자 요약과 최근 세션 흐름을 분리
- 결과 화면 CTA를 짧게 줄이고 틀린 문제 재학습 흐름을 더 바로 보이게 조정
- `.gitignore`에 `.omx/`와 환경 흔적 제외 규칙 추가
- GitHub Actions 기반 GitHub Pages 배포 워크플로 추가

### Notes
- `3자리 덧셈`은 현재 구현에서 `받아올림 없는 기본형`으로 고정되어 있다.
- 받아내림 1번은 규칙 정의 기준으로 `일의 자리만 받아내림` 패턴으로 고정 구현했다.
- 생성 후 검증을 통과한 문제만 실제 세션에 사용한다.
